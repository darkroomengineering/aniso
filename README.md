## Aniso

An open-source ASCII tool built by darkroom.engineering and designed by Studio Freight to generate and customize character-based imagery.

<br/>

## Features

- Granularity control
- Character limit
- Character picker
- Image color overwrite
- Image drop support
- Exporter
